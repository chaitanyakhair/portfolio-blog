## What is PostgreSQL?

[PostgreSQL](https://www.postgresql.org/) (also known as **Postgres**) is an advanced, enterprise-class, **open-source relational database** that‚Äôs been in active development for over 30 years.

It's known for:

- **ACID compliance**
- **Robust data integrity**
- **Rich SQL support**
- **Extensibility** (custom types, functions, etc.)

## Easy to Connect with Any Backend

PostgreSQL works beautifully with almost any backend language and framework.

### Node.js (with Prisma or Sequelize):

```bash
npm install prisma --save-dev
npx prisma init
```

In your .env:
```bash
DATABASE_URL="postgresql://user:password@localhost:5432/mydb"
```

Then:
```bash
npx prisma db push
```
You‚Äôre connected with PostgreSQL!

### Python + Django:
```bash
pip install psycopg2
```
In settings.py:
```bash
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'mydb',
        'USER': 'user',
        'PASSWORD': 'password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
That‚Äôs it. Django is now using PostgreSQL.

### NestJS + TypeORM:
```bash
npm install @nestjs/typeorm typeorm pg
```
In your AppModule:
```bash
TypeOrmModule.forRoot({
  type: 'postgres',
  host: 'localhost',
  port: 5432,
  username: 'user',
  password: 'password',
  database: 'mydb',
  autoLoadEntities: true,
  synchronize: true,
});
```
Done. Now your Nestjs is connected with PostgreSQL.
